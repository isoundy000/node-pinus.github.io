  • afterFilter(err: Error, routeRecord: RouteRecord, msg: any, session: FrontendOrBackendSession, resp: any, cb: HandlerCallback): void
    • Defined in common/service/filterService.ts:90

    Do after filter chain. Give server a chance to do clean up jobs after request responsed. After filter can not change the request flow before. After filter should call the next callback to let the request pass to next after filter.

      error object

    • routeRecord: RouteRecord

      session object for current request

    • resp: any

      response object send to client

    • cb: HandlerCallback

      cb(err) callback function to invoke next chain node
